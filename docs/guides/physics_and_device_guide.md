**最后更新**: 2026-04-13

## 概述

EFD-PINNs 实现了电润湿显示的核心物理约束，确保模型预测的物理合理性。本项目采用"有效物理参数"校准策略，使 PINN 的宏观响应与实验曲线高度一致。

详细参数说明请参见[物理理论与器件规格指南](physics_and_device_guide.md#physics-parameters)。

## ⚡ 核心物理方程

### 1. Young-Lippmann 方程（已校准）

描述接触角与电压的关系：

```
cos(θ) = cos(θ₀) + ε₀εᵣ(V-V_T)²/(2γd)
```

**物理意义**:
- 电压增加 → 接触角减小
- 接触角减小 → 油墨被排开 → 开口率增加
- 阈值电压 V_T≈3V 以下几乎无响应

### Young-Lippmann 方程参数

```python
theta0 = 120.0        # 初始接触角 (度)
epsilon_r = 12.0      # 四层串联等效: SU-8+Teflon AF+Oil+Polar liquid
epsilon_h = 1.9       # Teflon 疏水层
gamma = 0.015         # 有效表面张力 (N/m)（已校准）
d_dielectric = 4e-7   # SU-8 厚度 (m) = 400nm
d_hydrophobic = 4e-7  # Teflon 厚度 (m) = 400nm
V_threshold = 3.0     # 阈值电压 (V)
epsilon_0 = 8.854e-12 # 真空介电常数 (F/m)
```

### 工作电压

| 参数 | 值 |
|------|-----|
| 阈值电压 | **3V** |
| 工作电压范围 | 0-30 V |
| 典型工作电压 | 20 V |
| 翻墙临界电压 | >20 V |

### 接触角变化（已校准）

| 电压 (V) | 接触角 (°) | 变化量 (°) |
|---------|-----------|-----------|
| 0 | 120.0 | 0.0 |
| 6 | ~119 | ~1 (开始响应) |
| 10 | 119.2 | 0.8 |
| 20 | 115.2 | 4.8 |
| 30 | 108.2 | 11.8 |

### 2. 二阶欠阻尼响应

描述接触角的动态响应（升压过程）：

```
θ(t) = θ_eq + (θ₀-θ_eq)·e^(-ζω₀t)·[cos(ω_d·t) + ζ/√(1-ζ²)·sin(ω_d·t)]
```

**参数（已校准）**:
| 参数 | 符号 | 值 | 说明 |
|------|------|-----|------|
| 时间常数 | τ | 5 ms | 电润湿响应速度 |
| 阻尼比 | ζ | 0.8 | 欠阻尼，小超调 |

### 二阶欠阻尼系统

```python
tau = 5e-3    # 时间常数 (5ms)
zeta = 0.8    # 阻尼比 (欠阻尼)

omega_0 = 1 / tau  # 自然频率 = 200 rad/s
omega_d = omega_0 * np.sqrt(1 - zeta**2)  # 阻尼频率 = 120 rad/s
```

### 预期响应特性

| 参数 | 值 | 说明 |
|------|-----|------|
| 响应时间 (t₉₀) | ~14 ms | 达到90%稳态 |
| 超调 | ~4% | zeta=0.8 时 |
| 稳定时间 | <20 ms | 进入±2%误差带 |

### 3. 两相流方程 (PDEs)

#### 连续性方程 (Continuity)
```
∇·u = 0
```
确保流体质量守恒（不可压缩流体）。

#### VOF 方程 (Interface Tracking)
```
∂φ/∂t + u·∇φ = 0
```
在 6D Triad 空间中，$\partial/\partial t$ 对应 $\partial/\partial t_{since}$。追踪油墨-极性液体界面。

#### Navier-Stokes 方程 (Momentum)
```
ρ(∂u/∂t + u·∇u) = -∇p + μ∇²u + F_st
```
其中 $F_{st}$ 是表面张力项（采用 CSF 连续表面力模型）。

## 📊 φ 场与开口率（已校准）

### φ 场物理定义

| φ 值 | 物理含义 |
|------|----------|
| φ = 1 | 纯油墨（不透明）|
| φ = 0 | 纯极性液体（透明区域）|
| 0.1 < φ < 0.9 | 界面过渡区 |

**开口率定义**:
```python
# η = (φ < 0.5) 在底面 z=0 投影的面积比例
# 实际上是通过对 φ 进行积分计算：
eta = (1.0 - torch.mean(phi[z=0])).item()
```

### 开口率映射参数

```python
k = 1.0              # 陡度参数
theta_scale = 30.0   # 角度缩放因子
alpha = 0.02         # 电容反馈强度

```

## 🔬 物理机制

### 电润湿机制

1. **电润湿作用在极性液体上**（不是油墨）
2. **极性液体铺展**，将油墨从像素中心挤向边缘/角落
3. **油墨亲疏水层**（底部 Teflon），不亲围堰壁（相对亲水）
4. **油墨贴底收缩**，形成液滴，不会主动爬墙
5. **翻墙条件**：20V 以上油墨被挤压到极限可能翻墙

### 油墨分布模式

| 开口率 | 模式 | 描述 |
|--------|------|------|
| η < 50% | 中心开口 | 油墨环绕中心透明区域 |
| η > 50% (早期) | 四角液滴 | 毛细管切断，油墨分布在四角 |
| η > 50% (稳态) | 单角液滴 | 油墨汇聚到一个角落 |

## 边界条件 (Boundary Conditions)

### 底面 (z=0)
接触角边界条件：
```
∂φ/∂z = |∇φ| · cos(θ(V, t_since))
```
- **升压**: θ 遵循二阶响应曲线。
- **降压**: θ 瞬间恢复至 120°。

### 侧壁 (Pixel Walls)
- **无滑移条件**: u = 0
- **像素墙接触角**: θ_wall ≈ 70° (亲水性)

采用 **60,000 Epochs** 渐进式加载：

| 约束项 | 初始权重 | 最终权重 | 说明 |
|------|------|------|------|
| **Initial Condition** | 100.0 | 100.0 | t=0 静态分布 |
| **Boundary (z=0)** | 500.0 | 500.0 | 接触角约束 (关键) |
| **Continuity** | 0.0 | 0.5 | 质量守恒 |
| **VOF** | 0.0 | 0.5 | 界面演化 |

## 使用示例

### 1. 基础使用

```python
from src.models.aperture_model import EnhancedApertureModel

# 创建模型（使用默认配置）
model = EnhancedApertureModel()

# 预测开口率
result = model.predict(voltage=20.0)

print(f"电压: {result['voltage']:.1f}V")
print(f"接触角: {result['theta']:.1f}°")
print(f"开口率: {result['aperture_percent']:.1f}%")
print(f"开口半径: {result['r_open_um']:.1f}μm")
```

### 2. 批量预测

```python
import numpy as np

voltages = np.linspace(0, 30, 31)  # 0V 到 30V
results = [model.predict(v) for v in voltages]

# 提取结果
apertures = [r['aperture_percent'] for r in results]

print(f"0V 开口率: {apertures[0]:.1f}%")
print(f"30V 开口率: {apertures[-1]:.1f}%")
```
