## 根本原因

### 原因1：训练只测试3个电压，且时间不一致（主要）

**代码位置**: `src/models/pinn_two_phase.py:1809`

```python
tests = [(10.0, 0.010), (20.0, 0.010), (30.0, 0.025)]
```

**问题**：
1. 只测试3个电压 (10V, 20V, 30V)
2. 30V使用 t=0.025s，其他用 t=0.010s
3. 验证时用 t=0.02s 测试所有电压

**后果**：
- 0V、25V 等问题电压**没有被训练约束**
- 训练损失和验证指标不可比

### 原因2：权重太小（次要）

**体积守恒权重**：
```python
base_weight = 50.0
```

**界面数据权重**：
```python
base_weight = physics_cfg.get("interface_weight", 500.0)
```

**对比**：数据损失权重500，体积守恒权重50，相差10倍。

### 原因3：采样点太少

**采样点数**：
```python

```

3375个点对于3D空间太稀疏，体积积分不准确。

## 第一性原理推导

### 体积守恒的数学表达

在VOF方法中，油墨体积定义为：
```
V_ink(t) = ∫_Ω φ(x, t) dV
```

体积守恒要求：
```
dV_ink/dt = 0
```

推导：
```
dV_ink/dt = d/dt ∫_Ω φ dV
         = ∫_Ω (∂φ/∂t) dV  (假设域边界固定)
         = ∫_Ω (-u·∇φ) dV  (用VOF输运方程 ∂φ/∂t + u·∇φ = 0)
         = ∫_Ω (-∇·(φu) + φ∇·u) dV  (乘积法则)
         = -∫_∂Ω φ(u·n) dS + ∫_Ω φ(∇·u) dV  (散度定理)
```

因此，体积守恒要求：
```
∫_∂Ω φ(u·n) dS - ∫_Ω φ(∇·u) dV = 0
```

### 关键洞察

**VOF输运方程 `∂φ/∂t + u·∇φ = 0` 本身不保证体积守恒**！

要保证体积守恒，需要：
1. 边界条件：`u·n = 0`（无通量）
2. 或者：`∫_Ω φ(∇·u) dV = 0`（φ加权的连续性）

**当前代码的问题**：
- 使用了全域连续性 `∇·u = 0`（太强）
- 但没有φ加权（不对）

## 数值误差来源

### 自动微分的误差

PINN使用自动微分计算梯度：
```python
# 计算散度
u_x = torch.autograd.grad(u.sum(), x)[0][:, 0]
v_y = torch.autograd.grad(v.sum(), x)[0][:, 1]
w_z = torch.autograd.grad(w.sum(), x)[0][:, 2]

div_u = u_x + v_y + w_z
```

**问题**：
1. 自动微分有截断误差
2. 神经网络逼近有误差
3. 优化器只让损失降低，不保证严格满足等式

**结果**：`div_u ≈ 0.1`（而不是精确的0）

### 边界条件的问题

**当前实现**（`pinn_two_phase.py:1185`）：
```python
bc_values.append([0.0, 0.0, 0.0, 0.0, phi])
#             u    v    w    p    φ
```

**侧面（x=0, x=Lx, y=0, y=Ly）**：`u = v = w = 0`（无滑移）⚠️ 应该是 `u·n = 0`（无通量）

**但边界条件不是体积守恒误差的主要来源**！

## 明确的解决方案

### 修改1：增加测试电压（必须）

**文件**: `src/models/pinn_two_phase.py`
**位置**: 第1809行

```python
# 修改前
tests = [(10.0, 0.010), (20.0, 0.010), (30.0, 0.025)]

# 修改后
tests = [(0.0, 0.020), (5.0, 0.020), (10.0, 0.020),
         (15.0, 0.020), (20.0, 0.020), (25.0, 0.020), (30.0, 0.020)]
```

**理由**：
- 覆盖所有问题电压
- 使用统一的时间 t=0.020s（与验证一致）
- 7个测试点足够覆盖电压范围

### 修改2：增加权重（必须）

**位置**: 第1822行

```python
# 修改前
base_weight = 50.0

# 修改后
base_weight = 500.0  # 与界面损失权重相同
```

**理由**：让体积守恒约束与数据拟合同等重要。

### 修改3：增加采样点（推荐）

**位置**: 第1798行

```python
# 修改前
n_vol = 15  # 3375个点

# 修改后（方案A：均匀网格）
n_vol = 50  # 125000个点

# 修改后（方案B：蒙特卡洛采样）
n_vol = 50000
x = torch.rand(n_vol, device=self.device) * Lx
y = torch.rand(n_vol, device=self.device) * Lz
z = torch.rand(n_vol, device=self.device) * Ly
pts_base = torch.stack([x, y, z], dim=1)
```

**理由**：更准确的体积积分。

### 修改4：添加φ加权连续性（可选，增强）

**文件**: `src/physics/constraints.py`

```python

    'continuity': continuity,
    'phi_weighted_continuity': phi**2 * continuity,  # 新增
    ...

```

