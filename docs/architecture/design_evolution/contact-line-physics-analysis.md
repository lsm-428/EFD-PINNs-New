# 接触线物理验证分析

EFD-PINNs 中接触线（contact line）行为的物理自洽性验证。以下分析基于简化的 1D 解析推导，用于核验模型参数的物理合理性。

---

## 1. 接触线钉扎（Sidewall Pinning）

**场景**: V=0, 油膜平铺, 接触线在像素边缘 x≈0。

Young-Lippmann 平衡接触角 θ_eq = 71°，即 cos θ_eq ≈ 0.326。模型通过 `sidewall_contact_angle` 约束强制 n̂·n̂_wall = cos θ_eq。

**三种区域的物理预期**:

| 区域 | 位置 | 预期行为 |
|------|------|----------|
| 边缘钉扎区 | z=0, x≈0 | 高约束 loss，cos θ_local → cos θ_eq |
| 边缘上方 | z=3μm, x≈0 | 非钉扎区，正常流场 |
| 底面中心 | z=0, x=Lx/2 | 非边缘，无接触线约束 |

**钉扎阈值**: 当 |cos θ_local − cos θ_eq| > 0.15 时判定为"突破"状态。cos θ_eq(0V) = cos(71°) ≈ 0.326，平铺状态下 cos θ_local ≈ −1（θ ≈ 180°完全铺展），Δcos ≈ 1.326 > 0.15，因此 V=0 时应处于"突破"状态——但钉扎力仍然显著。

---

## 2. 受控滑移（Controlled Slip）

滑移速度模型: **v_slip = α · Δcos**，其中 α 为滑移系数，Δcos = cos θ_eq − cos θ_local。

**1D 解析**: 对于平衡剖面 φ(z) = 0.5(1 − tanh((z−z₀)/ε)):
- φ_z = −2φ(1−φ)/ε
- |∇φ| = 2φ(1−φ)/ε
- cos θ_local = φ_z / |∇φ| = −1（完全铺展极限）

**参数验证**:

| 参数 | 值 | 含义 |
|------|-----|------|
| α | 0.1 | 滑移系数 |
| cos θ_eq (V=0) | cos(120°) ≈ −0.5 | 疏水平衡态 |
| cos θ_local (平铺) | −1.0 | 完全铺展 |
| Δcos | 0.5 | 驱动力 |
| v_slip 期望 | α · Δcos = 0.05 | 无量纲滑移速度 |

**物理含义**:
- Δcos > 0 → 接触角需要减小（从 180° → 120°）
- v_slip > 0 → 接触线应向中心收缩（油膜开口/去浸润）
- 特征速度: v_slip × 2/ε ≈ 6.7×10⁴ m/s（基于 ε=1.5μm）

**收敛条件**: loss_slip = ‖v_slip − α·Δcos‖² → 0 时模型自洽。

---

## 3. 墨滴汇聚（Droplet Coalescence）

对比 CSF 表面张力与 AC 方程扩散力的竞争，判断墨滴是否趋向圆润。

**物理参数**:

| 参数 | 值 | 含义 |
|------|-----|------|
| σ | 0.02505 N/m | 物理表面张力（CSF） |
| ε_if | 1.5 μm | 界面宽度 |
| γ_AC | 4.5×10⁻⁷ | AC 迁移率 |

### 3.1 CSF 驱动力

对于半径 R=20μm 的液滴，Laplace 压力:
```
ΔP_CSF = σ / R = 0.02505 / 20×10⁻⁶ = 1252 Pa
```
该压力差驱动流场使液滴趋向圆形（表面能最小化）。

### 3.2 AC 扩散力

AC 方程扩散项 γ∇²φ 产生平滑力:
```
F_AC ≈ γ_AC / R = 4.5×10⁻⁷ / 20×10⁻⁶ = 0.0225 s⁻¹（无量纲）
```
远小于 CSF 的物理压力。

### 3.3 能量驱动

| 状态 | 面积 | 说明 |
|------|------|------|
| 平铺油膜 | 174×174 μm² = 3.03×10⁻⁸ m² | 初始状态 |
| 角落液滴（半球） | 2πR² ≈ 2.51×10⁻⁹ m² | 汇聚后 |

界面能差: ΔE = σ × ΔA ≈ 0.02505 × 2.78×10⁻⁸ ≈ 0.70 nJ。CSF 有足够能量驱动汇聚。

### 3.4 主导性结论

```
CSF 压力 / AC 扩散 ≈ 1252 / 0.0225 ≈ 5.6×10⁴ ≫ 100
```

✅ **CSF 主导界面形态** — 墨滴应向圆润形状演化。

⚠️ 如果训练后墨滴仍然糊在角落，检查 CSF 权重是否被 loss 调度过度稀释。
